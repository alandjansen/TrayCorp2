TrayCorp

Um CRUD de produtos


Com este projeto pode-se cadastrar, visualizar, editar e deletar produtos. Temos na lista de produtos a ordenação por ID e por nome e a busca de produto por nome.


Conteúdo


  1. Controllers

    1. HomeController.cs - criado junto com o projeto
    2. ProdutosController.cs - controller aonde é feita as interações dos produtos


  2. Models

    1. Produto.cs - modelo da tabela PRODUTO, não utilizado
    2. ProdutoDBContext.cs - modelo de acesso ao banco de dados, no caso acesso ao .edmx
    3. ProdutoViewModel.cs - view utilizada na exibição da tabela PRODUTO na qual contém as validações de conteúdo e os títulos


  3. Repositorio

    1. TrayCorp.edmx - entity do banco com a tabela PRODUTO


  4. Views

    1. Produtos

      1. Create.cs - usada na criação de novo produto
      2. Delete.cs - usada para deletar um produto
      3. Details.cs - usada para exibir detalhes de determinado produto
      4. Edit.cs - usada para editar um produto
      5. Index.cs - usada na exibição e pesquisa da lista de produtos
